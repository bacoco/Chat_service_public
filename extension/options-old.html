How to get your Albert API key:

  1. Visit albert.api.etalab.gouv.fr
  2. Click "S'inscrire" (Sign up) to create a free account
  3. Once logged in, go to the "API Keys" section
  4. Generate a new API key
  5. Copy and paste it below

API Configuration

Your API key is stored securely in your browser and never sent anywhere except to Albert's API.

About

Universal Web Assistant uses Jina AI to extract page content and Albert LLM (French government AI) to provide intelligent responses. Your API key enables direct communication with Albert's servers.
